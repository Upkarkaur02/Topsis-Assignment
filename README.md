Topsis-UpkarKaur-102497017
1. Methodology
The package implements the Technique for Order of Preference by Similarity to Ideal Solution (TOPSIS) for multi-criteria decision making
2. Description
   
This project is divided into three parts:

Part I: Command Line Tool: A standalone script for ranking alternatives based on weights and impacts.

Part II:A Python package is developed and hosted on PyPI following specific naming conventions.

-Package Name Format: Topsis-UpkarKaur-102497017

-User Manual: Included in the package documentation.

-Testing: Can be installed via pip install <package-name> and run directly from the terminal.

Part III: Web Service:A Flask-based web interface allows users to upload files and receive results via email.

Web Interface Fields

-File Name: Browse and upload a .csv or .xlsx file.

-Weights: Comma-separated weights (e.g., 1,1,1,1).

-Impacts: Comma-separated signs (e.g., +,+,-,+).

-Email Id: Valid email address to receive the result.xlsx attachment.
